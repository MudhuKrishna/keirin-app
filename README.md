# Keirin - Software Consulting & Gaming Services Website

A modern single-page React application built with **TypeScript** showcasing Keirin's services in software consulting, staff augmentation, and gaming solutions.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern UI**: Clean, professional interface with gradient styling
- **Smooth Navigation**: Fixed navbar for easy navigation between sections
- **Service Sections**:
  - Why Keirin Stands Out - Core values and differentiators
  - Software Consulting Services - Detailed consulting offerings
  - Staff Augmentation - Flexible staffing solutions
  - Gaming Services - Next-gen gaming technology solutions

## Installation & Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm start
   ```
   The app will open at `http://localhost:3000`

3. **Build for production**:
   ```bash
   npm build
   ```

## Available Scripts

- `npm start` - Runs the development server
- `npm test` - Runs the test suite
- `npm build` - Creates an optimized production build
- `npm eject` - Exposes the build configuration (cannot be undone)

## Technologies Used

- **React 18** - JavaScript library for building user interfaces
- **TypeScript** - Typed superset of JavaScript for type safety and better development experience
- **CSS3** - Styling with gradients and responsive design
- **React Scripts** - Build tools and development server

## Browser Support

Works in all modern browsers including:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

All content is easily editable in the component files. To update:
- **Text content**: Edit the JSX in component files
- **Colors/Styling**: Modify the CSS files in `/src/styles`
- **Layout**: Adjust grid and flexbox properties in CSS
